# R-code-Rfinance-Chicago-2023


1. Open project R_Finance
2. Load slides_R_finance.R into project
3. R-code is based on two different parts/files: 
	a. first part is based on teaching (ATSF-course)
	b. second part is based on XAI-paper
4. Slides of presentation are in folder ‘Slides presentation’
5. R functions are in folder ‘R’
6. Data is in folder ‘Data’
7. Previously stored results are in folder ‘results’ (can be replicated by R-code) 

There are some illustrations/examples for both LPD and QPD in the R-folder (look at ‘play with LPD’ and ‘play with QPD’ R-files)
